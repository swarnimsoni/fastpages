Visualization

Fastpages just copies visualizations/charts/graphs from Jupyter notebook, however, only altair charts remain interactive in fastpages, others become static.

Here are some random charts with different libraries.

with Altair 

%%capture
#install altair package
!pip3 install cite2c altair vega_datasets

import altair as alt
from vega_datasets import data

source = data.cars()

alt.Chart(source).mark_circle(size=60).encode(
    x='Horsepower',
    y='Miles_per_Gallon',
    color='Origin',
    tooltip=['Name', 'Origin', 'Horsepower', 'Miles_per_Gallon']
).interactive()

with pyplot
